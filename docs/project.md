## Problem

Cooling = **30–40%** of DC power; similar in cold chain and buildings. Fixed BMS logic reacts after heat builds. **India:** 45°C summers, costly grid, DC + cold-chain boom — mid-market has **no optimiser** (Phaidra chases hyperscalers).

---

## Solution

1. **Predict** — rack power + weather + thermal mass; act before spike  
2. **Optimise** — MPC/RL inside hard safety band  
3. **Operate by voice** — no dashboard training  
4. **Prove on twin first** — twin → shadow → advise → control
